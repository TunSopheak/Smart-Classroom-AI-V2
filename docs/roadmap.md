# Roadmap

## Phase 1: Clean Foundation

- FastAPI app structure.
- SQLite development database.
- SQLAlchemy models.
- Dashboard page.
- Architecture documentation.

## Phase 2: Academic Management

- CRUD for teachers, subjects, class groups, students, and enrollments.
- Weekly schedule management.
- Validation for schedule conflicts.

## Phase 3: Sessions and Attendance

- Automatic session generation from weekly schedules.
- Manual attendance records.
- QR attendance workflow.
- Attendance summaries.

## Phase 4: AI Monitoring

- Camera input using OpenCV.
- Face detection.
- Attention detection.
- Phone usage detection.
- AI event logging linked to sessions.

## Phase 5: IoT and Reports

- Device status tracking.
- Attendance reports.
- AI event reports.
- PDF generation with ReportLab.
